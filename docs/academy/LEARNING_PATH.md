# Mini BOP Academy — Learning Path

> Roadmap de aprendizagem recomendado para novos colaboradores e estudantes.

---

# Objetivo

Este documento apresenta a sequência sugerida de estudos da Mini BOP Academy.

Embora os módulos possam ser consultados individualmente, seguir esta ordem proporciona uma evolução gradual dos conceitos.

---

# Trilha de Aprendizagem

```text
Iniciante
    │
    ├── 00 - Welcome

    ├── 04 - Oracle Core

Avançado
```

# Perfis recomendados

Prioridade:

- Oracle Core
- Batch Pipeline
- Performance
- Recovery

## Data Engineer

Prioridade:

- Batch Pipeline

## Arquiteto de Software

Recomenda-se percorrer toda a Academy na ordem proposta.

---

# Resultado esperado

Ao concluir a trilha o leitor deverá ser capaz de:

- compreender a arquitetura do Mini BOP;
- navegar pelo código com segurança;
- explicar os principais conceitos utilizados no projeto;
- relacionar Oracle com tecnologias modernas de Data Engineering.
