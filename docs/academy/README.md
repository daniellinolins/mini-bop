# Mini BOP Academy

> Guided onboarding path for the Mini BOP project.

The Academy complements the technical documentation by providing a structured learning experience. It starts with financial and architectural fundamentals and gradually moves toward Oracle core processing, governance, Big Data and engineering decisions.
