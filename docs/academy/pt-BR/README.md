# Mini BOP Academy (PT-BR)

> Onboarding técnico oficial do projeto Mini BOP.

---

## Bem-vindo

Esta Academy foi criada para acelerar o entendimento do projeto utilizando uma abordagem progressiva.

Ela complementa a documentação técnica tradicional e explica:

- o domínio financeiro;
- a arquitetura Oracle;
- o pipeline Batch;
- os conceitos de Governança de Dados;
- a evolução para Big Data e Analytics.
