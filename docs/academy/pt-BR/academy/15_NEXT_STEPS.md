# Como utilizar o Mini BOP

O projeto pode ser utilizado para:

- estudos individuais;
- onboarding de novos colaboradores;
- preparação para entrevistas técnicas;
- demonstração de padrões arquiteturais;
- experimentação de evoluções em Engenharia de Dados.

# Contribuições

Ao evoluir o projeto:

- mantenha a documentação sincronizada com o código;
- registre decisões arquiteturais relevantes;
- documente novos Débitos Técnicos;
- preserve a organização da Academy.
