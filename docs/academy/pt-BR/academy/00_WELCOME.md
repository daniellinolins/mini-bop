# Módulo 00 — Bem-vindo à Mini BOP Academy

## Objetivo

Esta Academy foi criada para reduzir o tempo de onboarding de novos desenvolvedores e engenheiros de dados.

Ela não substitui o README do projeto. Em vez disso, complementa a documentação oficial explicando:

- o domínio financeiro necessário para compreender o projeto;
- a arquitetura Oracle implementada;
- o pipeline de processamento;
- as decisões de engenharia adotadas;
- a evolução para arquiteturas modernas de Data Engineering.

---
