# Roadmap

Mini BOP was delivered incrementally through structured platform phases.

## Major Milestones

| Phase Group | Scope |
|---|---|
| Oracle Foundation | Core schema, batch control, trade load and transformation. |
| Operational Controls | Logging, restartability, recovery, reconciliation and data quality. |
| Performance Engineering | Bulk loading, partitioning and parallel processing. |
| Hadoop Integration | Oracle-to-HDFS export and landing zone creation. |
| Hive Layer | External table and query layer. |
| Spark Layer | Distributed processing and Spark SQL analytics. |
| Incremental Processing | Current-state, history, change logs and checkpointing. |
| Orchestration | Airflow end-to-end DAG. |
| Monitoring | Health checks, reports and operational visibility. |
| API & Dashboard | REST API, Swagger and operational dashboard. |
| Local Platform Packaging | Repeatable local startup and validation. |
| Demo Readiness | Bootstrap, documentation and presentation assets. |

## Current Status

The platform currently includes:

- Oracle source and processing logic.
- HDFS landing and analytical storage.
- Hive external table layer.
- Spark processing and analytics.
- Incremental processing and checkpoints.
- Airflow orchestration.
- Monitoring reports.
- FastAPI integration.
- Dashboard.
- Bootstrap scripts.
- Professional documentation baseline.
